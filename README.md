# Student-Management-System-Java
A Java console-based application that allows users to:

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Store Marks
- Calculate Grades

Technologies Used:
- Java
- IntelliJ IDEA

Concepts Used:
- Classes & Objects
- Constructors
- Arrays
- Loops
- CRUD Operations
- Conditional Statements
